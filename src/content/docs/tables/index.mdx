# Tables

ImprintCSS provides a few options for quick-and-easy table styling:

* We apply a [default look to tables](/tables/default-table). (This can be explicitly styled as our [modern table style](/tables/modern-table) if you need to be explicit.)
* You can [disable ImprintCSS's default table styling](/tables/unstyled-table) if you wish.
* We offer a [grid-like table style](/tables/grid-table) if that's your thing.
