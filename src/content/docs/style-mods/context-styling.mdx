## Alert / Notification Colors

CSS Suffix  | Description
------------|-------------
`--info`    | Adds blue colors to the block.
`--success` | Adds green colors to the block.
`--warning` | Adds yellow colors to the block.
`--danger`  | Adds red colors to the block.

Always remember to add some form of visual icon too in your markup!

## Activation Colors

SOMETHING HERE

CSS Suffix    | Description
--------------|------------
`--disabled`  | Styles the block as-if no-one can interact with it.
`--faded`     | Styles the block so that it fades into the background.
