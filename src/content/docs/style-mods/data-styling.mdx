# Data Style Mods

## Striping Sibling Elements



CSS Suffix | Description
-----------|------------
`--striped` or `--striped-even` | Apply a different background to every second, fourth, sixth (and so on) element.
`--striped-odd` | Apply a different background to every first, third, fifth (and so on) element.
