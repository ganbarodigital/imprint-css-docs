# Block Styling

## Border Styling

SOMETHING HERE

CSS Class         | Description
------------------|------------
`imprint-alert`   | Applies a background to any contained header inside your `div`.
`imprint-block`   | Applies a square border to your `div`.
`imprint-callout` | Apples a border to just the left side of your `div`.
`imprint-panel`   | Applies a rounded border to your `div`.

Each of these can be combined with the [context classes](./context-styling) to style the border to suit.

### Examples

Here is an example of a paragraph before the example, to show how the flow spacing works.

<div class="imprint-alert --shaded">
    <h4>This Is A Heading Inside An Alert</h4>
    <div class="imprint-inner">
        <p>This is content inside the alert.</p>
        <p>This is more content inside the alert. Notice how these paragraphs left-align with the heading text above? I think that's an aesthetically-pleasing choice. It also makes it a <em>lot</em> easier to combine `border--alert` with other border styling choices.</p>
    </div>
</div>
