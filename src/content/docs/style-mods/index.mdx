# Style Mods

## What Are Style Mods?

"Style mods" are a way to add an additional look to an HTML tag. If you like, they modify the CSS style (hence "style mods" ðŸ™‚).

All style mods have a `--<style-mod>` suffix (e.g. `table--striped`).
