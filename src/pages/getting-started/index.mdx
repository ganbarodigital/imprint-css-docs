# Getting Started

ImprintCSS is a plugin for [Tailwind CSS](https://tailwindcss.com).

To use it:

1. `npm require @stuartherbert/imprint-css`
2. add `@stuartherbert/imprint-css` as a plugin to your `tailwind.config.js` file

You should be good to go.
